## Installation
This projects runs on Python 3.9 with the following libraries:

- OpenCV (opencv-python)
- mediapipe
- pydub
- RPi.GPIO
- numpy

```
pip install opencv-python
pip install mediapipe
pip install pydub
pip install RPi.GPIO
pip install numpy
```
